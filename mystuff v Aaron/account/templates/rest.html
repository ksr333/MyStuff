REST API Example


            Endpoint: http://dithers.cs.byu.edu/iscore/api/v1/charges
            Valid credit card: Visa, 4732817300654, Exp. 10/14, CVC 411, Cardholder Name: Cosmo Limesandal

            Sample Charge Creation:
            curl http://dithers.cs.byu.edu/iscore/api/v1/charges \
            -d apiKey=YOUR_API_KEY \
            -d currency=usd \
            -d amount=5.99 \
            -d type=Visa \
            -d number=4732817300654 \
            -d exp_month=10 \
            -d exp_year=14 \
            -d cvc=411 \
            -d "name=Cosmo Limesandal" \ -d "description=Charge for cosmo@is411.byu.edu"

            Query Charges: curl --get http://dithers.cs.byu.edu/iscore/api/v1/charges \
            -d apiKey=YOUR_API_KEY
